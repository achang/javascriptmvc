Unit tests

Unit tests are used to verify your classes and methods are behaving as you expect. Here's how to get started.

  1. Create a 'test/unit' directory on the same level as your 'jmvc' directory.
  2. Create a file like 'animal_unit_test.js' and add the following code: 
    new $MVC.Test.Unit('animal',{
   test_truth: function() {
      this.assert(true);
   }
});
  3. Create a file called 'YOURAPPNAME_test.js' in your 'app' directory and add the following code: include('../test/unit/animal_unit_test.js')
  4. Reload your page.

For more information, check out the documentation.

Functional tests

Functional tests are used to simulate user interactions with your page. Here's how to get started.

  1. Create a 'test/functional' directory on the same level as your 'jmvc' directory.
  2. Create a file like 'widget_functional_test.js' and add the following code: 
    new $MVC.Test.Functional('widget',{
   test_truth: function() {
      this.assert(true);
   }
});
  3. Create a file called 'YOURAPPNAME_test.js' in your 'app' directory and add the following code: include('../test/functional/widget_functional_test.js')
  4. Reload your page.

For more information, check out the documentation.
